# Africa RCT E156 Portfolio

This umbrella repo ties together the four topic-specific public repos generated from the Africa-site ClinicalTrials.gov scan. All four active paper releases now use strict E156 format: one validated 156-word body plus companion artifacts.

## Included Topic Repos

- `malaria-e156`: Lean Africa-site malaria RCTs as a transfer template
- `hiv-e156`: Lean Africa-site HIV RCTs as a transfer template
- `hypertension-e156`: Lean Africa-site hypertension RCTs as a transfer template
- `maternal-health-e156`: Lean Africa-site pregnancy-related RCTs as a transfer template

## Active Release Model

- each topic repo has a validated `156`-word E156 body
- note blocks, metadata, and HTML companions sit outside the fixed body length
- long-form journal expansion is deferred unless explicitly requested

## Included Portfolio Files

- `TOPIC_MATRIX.md`: side-by-side topic snapshot
- `data/topic_comparison.md`: copied comparison note
- `MANIFEST.json`: local repo paths and key metrics
